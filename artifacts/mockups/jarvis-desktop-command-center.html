      Command Surfaces

      Supervisory verbs, not a chat prompt pretending to be structure.

      Silent Steering
      Review + Triage

      Pull the highest-signal items forward without expanding the whole system into noise.

      Inspect + Compare

      Open evidence, compare options, and understand consequences before acting.

      Redirect + Bound

      Change lane direction, narrow scope, or impose quieter execution without losing continuity.

      Approve + Defer

      Release, postpone, or hold bounded actions without context switching into another surface.

      Desktop Versus Mobile

      Desktop is for supervision and continuity recovery. Mobile is for interruption-safe quick action.

      Desktop
      • Parallel lane awareness
      • Evidence inspection
      • Decision batching
      • Resume stack
      • Agent society visibility
      Mobile
      • Interrupt approvals
      • Voice capture
      • One-step defer or resume
      • Heads-up updates
      • Fast bounded commands

      Command Dock

      Always reachable. Short silent verbs for supervision, not a catch-all prompt box.
