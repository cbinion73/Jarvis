# Twin Wake slicer pack

- Family: custom-form
- Printer: creality-k2-pro-combo
- Profile: creative-concept-balanced
- Material: PLA
- Layer height: 0.20 mm
- Infill: 18% gyroid
- Supports: Tree or organic supports only where the silhouette truly needs them

Use the 3MF file as the preferred import when your slicer accepts it.
Fall back to the STL when you want to override slicer-side settings manually.

Constraints: Keep overhangs modest so it can be printed without heroic support. Maintain at least 3 mm thickness through the ribbon arms for strength on an FDM printer.
